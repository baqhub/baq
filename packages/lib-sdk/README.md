This is the official JavaScript SDK for the [BAQ](https://baq.dev) federated app platform. It provides abstractions and helpers that make it easier to work with BAQ records instead of directly calling the [HTTP APIÂ­](https://baq.dev/docs/learn/guides/using-the-http-api).

## Installation

The BAQ JavaScript SDK is distributed as an NPM package.

```bash
npm install @baqhub/sdk
```
